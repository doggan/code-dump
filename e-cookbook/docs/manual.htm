About

Introduction

E-Cookbook is an electronic cookbook, or recipe organizer, with a principle design goal of ease-of-use. It allows the user to create and maintain recipes that can be searched and sorted through. All recipes are saved out to a single file, which allows the user to easily backup their recipe collection.

Ideally, this application can replace the old tattered cookbook in your kitchen.

Features

Here are some of the features of E-Cookbook:

  • Cross-Platform
    • Windows and Mac OS X supported by the developer
  • Easy to use interface
  • Multiply ingredient amounts to any number of servings
  • Search through your recipes using your own custom filters
  • Sort recipes by any category
  • View/Print recipes in a nicely formatted manner
  • *Select a random recipe (Answers the common question: 'What to make for dinner?!')
  • *Find a recipe for the items you have in your refrigerator
* - Denotes a feature in a future release

Technical Notes

This software was written in Python. The user interface is written in wxPython. The database backend utilizes PySqlite. It has been tested primarily on Windows 98/2000/XP and Mac OS X, although it should work on all platforms that provide a Python interpreter.

E-Cookbook is free software released under the GPL. A copy has been provided (see: 'COPYING').

If you find a bug (or 'feature'), please report it to the Sourceforge project page (link at top).

If you find this software useful, please let me know! (email at top). Or if you really find it useful, donations are accepted through the Sourceforge project page.

Tutorials

General Notes

Remember to save your work often. You can save your cookbook at any time by going to the menu and selecting File > Save Cookbook.

Cookbooks are stored in a single file with a .cb extension. You can make copies of these files to back up your data. You can open an already created cookbook at a later time through the menu: File > Open Cookbook.

Getting Started

When you first open E-Cookbook, you are presented with a blank slate. The first thing you must do is create a cookbook to store your recipes in. Go to the menu. Select File > New Cookbook.

Now that a cookbook has been created, you can add new recipes. Click "Add Recipe". This opens the "Add New Recipe" dialog:

This dialog is split into 4 different tabs, which can be navigated to by clicking the tabs at the top of the window. The tab names are self explanatory.

On the "Description" tab, you can fill in all your recipes' fields. For the Category and Cuisine fields, default values are provided in a drop-down list. If you want to enter your own custom value, simply type it into the text field and press enter. This will make that value available in the drop-down list for any future item.

The next tab is "Ingredients". Again, the fields are self explanatory. For the Amount value, there are three boxes. These correspond to, from left to right, the whole number, numerator, and denominator of a mixed fraction of the form: 3 5/8. The following image shows how to enter a non-fractional value:

Once your ingredient properties are set, click Ok. You can add as many ingredients as you want. After an ingredient has been added, it will appear in the list view (shown below). List views are seen throughout this application to display information. They all behave in the same manner. To sort the items in the list view, click the column header (i.e. Name, Amt., Units, etc.).

You are now free to modify or delete ingredients. To modify, simply click an ingredient so that it is highlighted in the list view, and then click "Modify Ingredient". This will bring up the same dialog as seen earlier, but with the fields already filled in with the existing values.

Ingredients can also be deleted. Select all the ingredients to be deleted, and click "Delete Ingredient". You will be prompted with a confirmation box asking if you are sure you want to continue with the deletion process.

The "Instructions" and "Extra Notes" tabs are just text fields. Add whatever information you see fit here.

Click Ok to keep this newly added recipe. The recipe will now display in the main list view. You can now add/modify/delete recipes using the buttons on the left. They behave in exactly the same way as adding/modifying/deleting ingredients.

View Recipe

Once a recipe has been added, you can view it. Select the recipe you want to view, and Click "View Recipe". This opens the "View Recipe" dialog. This dialog shows all the information you have already entered in a read-only fashion (i.e. you can't edit anything from here).

There is one thing you can edit, though: Servings. If the number of servings was entered in the "Add/Modify Recipe" dialog, you can now scale that number of servings here. By changing the number of servings, the ingredient amounts will also be multiplied. You can click the "Reset Servings" button to reset the servings # to the original value. If you never entered the amount of servings for the given recipe (i.e. Servings = 0), this field will be grayed out.

Lastly on this dialog, you can Print or Print Preview the recipe by clicking on the buttons at the bottom of the dialog. Printing the recipe formats it in a nice manner, giving you a paper copy to be used as a quick reference.

Searching

Searching can be performed in the main dialog. Simply enter a keyword (filter) and select the field you wish to apply this filter to. As you type, the matching entries will be displayed in the list view. Perhaps the most useful category is search by "Ingredient Name". If you have some chicken you need to use up, enter "chicken" in the text field and "Ingredient Name" in the drop-down category field. All recipes that use chicken as an ingredient will be displayed.

Clear the searching criteria with the "Clear Search" button. Perform a "Case Sensitive" search by checking the check-box (off by default).
